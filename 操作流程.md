# RowColPage 操作流程

本文件整理 RowColPage 各版本的基本使用方式，並補充第三版目前完成後的實際操作流程。

## 1. 開啟系統

可依需求選擇本機版或雲端版：

- 第一版：<https://panggihsieh.github.io/RowColPage/>
- 第二版：<https://panggihsieh.github.io/RowColPage/v2/>
- 第三版：<https://panggihsieh.github.io/RowColPage/v3/>
- 第四版：<https://panggihsieh.github.io/RowColPage/v4/>

若要在本機執行：

```powershell
python -m http.server 8000
```

第三版建議在 `v3` 目錄單獨啟動：

```powershell
cd v3
python -m http.server 8002
```

第四版可直接用根目錄的靜態伺服器開啟：

```text
http://localhost:8000/v4/
```

## 2. 版本選擇

### 第一版

適合固定八格的基礎作業簿。

### 第二版

適合自學筆記、多格練習與可調整列欄版型。

### 第三版

適合逐題整理幾何題目，並保留右側書寫計算程序區。

### 第四版

適合從含有題號的圖片或 PDF 依題號規則自動抽題，並整理成可書寫的原題複習單。

## 3. 第三版設定流程

進入第三版後，可先設定：

- 標題
- 姓名
- 日期
- 起始編號
- 頁數
- 列數
- 每格輔助線
- 家長簽名顯示

第三版目前不提供欄數切換，版面固定為每題一列。

## 4. 第三版題圖帶入方式

每一題都可個別放入圖片，支援四種方式：

1. 點選 `上傳題圖`
2. 已有圖片時使用 `更換題圖`
3. 先複製截圖後，點選目標格再按 `Ctrl+V`
4. 點選 `貼上題圖`，直接從剪貼簿讀取圖片

補充：

- 題圖會自動縮放到左側題目區內
- 若該格已是目前貼上目標，畫面會有高亮提示
- 若不需要該題，可按 `清除`

## 5. 第三版預覽與列印

完成題圖與頁面設定後：

1. 檢查每題左側題圖是否正確
2. 檢查右側計算程序區是否保留足夠空白
3. 確認頁數、列數與編號是否正確
4. 點選 `列印 / 另存 PDF`

列印時會自動：

- 使用 A4 直式
- 保留題圖與計算程序區
- 隱藏上傳按鈕
- 隱藏題號

## 7. 目前結論

第三版與第四版目前都已完成可使用版本。第四版核心流程如下：

- 上傳含有題號的數學圖片或 PDF
- 可先用 `v4/example.pdf` 直接測試整條流程
- 設定題號規則與裁切範圍
- 自動抽題並開啟新的勾選頁
- 將勾選到的題目依序帶入左側題目區
- 視需要個別更換、貼上或清除
- 列印成可書寫的原題數學複習單
