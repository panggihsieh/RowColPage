# 八格簿操作流程

本文件說明「八格簿」第二版草稿的使用方式。

## 1. 開啟系統

可使用以下其中一種方式：

- 直接開啟 `index.html`
- 使用 localhost 開啟專案頁面

若使用 localhost，請在專案目錄執行：

```powershell
python -m http.server 8000
```

再於瀏覽器輸入：

```text
http://localhost:8000
```

## 2. 設定頁面內容

進入頁面後，可先調整上方控制區：

- 標題：輸入頁面標題，預設為「大南國小」
- 班級：輸入班級名稱，例如三甲
- 姓名：輸入學生姓名
- 日期：選擇當日日期
- 起始編號：設定第一格的起始數字
- 頁數：設定需要幾頁八格簿
- 每格輔助線：選擇格內輔助線樣式
- 家長簽名：切換是否顯示右上角簽名框
- 重設預設值：清除目前設定並回到預設狀態

## 3. 設定記憶功能

第二版新增設定記憶功能：

- 你修改過的標題、班級、姓名、日期、頁數與輔助線設定
- 會自動保存到目前瀏覽器
- 下次重新開啟同一台電腦與同一個瀏覽器時會自動帶回

如果要清除並回到預設內容，可按：

- `重設預設值`

## 4. 輔助線選擇

每格輔助線共有四種模式：

- 橫線
- 十字線
- 點
- 無

預設模式為：

- 十字線

## 5. 編號規則

每頁共有 8 格，排列方式如下：

- 4 列 2 欄
- 由左到右
- 由上到下

例如起始編號為 `1` 時，第一頁會依序產生：

```text
1  2
3  4
5  6
7  8
```

第二頁會接續下一組編號。

## 6. 頁首資訊列

第二版每頁上方新增資訊列：

- 班級
- 姓名
- 日期

如果班級或姓名沒有輸入，頁面上會顯示底線空格，方便手寫填入。

## 7. 家長簽名區

若開啟「家長簽名」功能：

- 頁面右上角會顯示「家長簽名」
- 後方會有一個長方形框線可供簽名

若關閉：

- 頁面右上角簽名區將隱藏

## 8. 頁碼與版本資訊

- 每一頁頁碼顯示在頁尾中央
- 格式為「第 X 頁」
- 頁碼上方會顯示版本資訊
- 目前版本為「第二版草稿 / 八格簿」

## 9. 列印或輸出 PDF

設定完成後，點選右上角：

- `列印 / 另存 PDF`

即可：

- 直接列印
- 另存為 PDF

建議列印設定：

- 紙張：A4
- 方向：直式
- 縮放：預設或 100%
- 邊界：依瀏覽器預設即可

## 10. 本版重點

這一版為「八格簿」第二版草稿，重點如下：

- 保留第一版的 A4 八格簿核心功能
- 新增班級、姓名、日期欄位
- 新增設定記憶功能
- 新增重設預設值功能
- 持續維持可列印與不溢頁的 A4 版型
