# 八格簿操作流程

本文件說明「八格簿」第一版的使用方式。

## 1. 開啟系統

可使用以下其中一種方式：

- 直接開啟 `index.html`
- 使用 localhost 開啟專案頁面

若使用 localhost，請在專案目錄執行：

```powershell
python -m http.server 8000
```

再於瀏覽器輸入：

```text
http://localhost:8000
```

## 2. 設定頁面內容

進入頁面後，可先調整上方控制區：

- 標題：輸入頁面標題，預設為「大南國小」
- 起始編號：設定第一格的起始數字
- 頁數：設定需要幾頁八格簿
- 每格輔助線：選擇格內輔助線樣式
- 家長簽名：切換是否顯示右上角簽名框

## 3. 輔助線選擇

每格輔助線共有四種模式：

- 橫線
- 十字線
- 點
- 無

預設模式為：

- 十字線

## 4. 編號規則

每頁共有 8 格，排列方式如下：

- 4 列 2 欄
- 由左到右
- 由上到下

例如起始編號為 `1` 時，第一頁會依序產生：

```text
1  2
3  4
5  6
7  8
```

第二頁會接續下一組編號。

## 5. 家長簽名區

若開啟「家長簽名」功能：

- 頁面右上角會顯示「家長簽名」
- 後方會有一個長方形框線可供簽名

若關閉：

- 頁面右上角簽名區將隱藏

## 6. 頁碼位置

- 每一頁頁碼顯示在頁尾中央
- 格式為「第 X 頁」

## 7. 列印或輸出 PDF

設定完成後，點選右上角：

- `列印 / 另存 PDF`

即可：

- 直接列印
- 另存為 PDF

建議列印設定：

- 紙張：A4
- 方向：直式
- 縮放：預設或 100%
- 邊界：依瀏覽器預設即可

## 8. 本版重點

這一版為「八格簿」第一版，重點如下：

- 可快速產生可列印八格簿
- 支援不同輔助線模式
- 支援家長簽名區顯示切換
- 適合國小教學與家庭作業使用
