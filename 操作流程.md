# 自學筆記操作流程

本文件說明「自學筆記」第二版草稿的使用方式。

## 1. 開啟系統

可使用以下其中一種方式：

- 直接開啟 `index.html`
- 使用 localhost 開啟專案頁面

若使用 localhost，請在專案目錄執行：

```powershell
python -m http.server 8001
```

再於瀏覽器輸入：

```text
http://localhost:8001
```

## 2. 設定頁面內容

進入頁面後，可先調整上方控制區：

- 標題：預設為 `大南六甲`
- 班級：目前保留在設定區
- 姓名：輸入學生姓名
- 日期：選擇日期
- 起始編號：設定第一格起始數字
- 頁數：設定產生幾頁
- 欄數：設定每頁欄數
- 列數：設定每頁列數
- 每格輔助線：切換格內輔助線樣式
- 家長簽名：切換第一頁右上角簽名區

## 3. 設定記憶功能

第二版新增設定記憶功能：

- 你修改過的設定會自動保存到目前瀏覽器
- 下次用同一台電腦、同一個瀏覽器打開時會自動帶回

如需清除並恢復預設設定，可點選：

- `重設預設值`

## 4. 每頁格數規則

第二版不再固定八格。

每頁格數由下面公式決定：

```text
欄數 × 列數 = 每頁總格數
```

例如：

```text
2 欄 × 4 列 = 8 格
3 欄 × 5 列 = 15 格
1 欄 × 2 列 = 2 格
```

## 5. 編號規則

編號方式固定如下：

- 左到右
- 上到下

例如 `2 欄 × 4 列`，起始編號為 `1` 時：

```text
1  2
3  4
5  6
7  8
```

下一頁會自動接續下一個編號。

## 6. 第一頁與後續頁差異

第二版固定規則如下：

### 第一頁顯示

- 標題
- 版型
- 姓名
- 日期
- 家長簽名

### 第二頁之後顯示

- 只有格子
- 只有頁碼
- 不再顯示頁首資訊

## 7. 家長簽名區

若開啟「家長簽名」功能：

- 第一頁右上角會顯示兩列文字
  - 家長
  - 簽名
- 右側會顯示簽名框

若關閉：

- 第一頁右上角簽名區不顯示

## 8. 頁碼格式

第二版頁碼顯示在頁尾中央，格式如下：

- `第1/2頁`
- `第2/2頁`

也就是：

```text
第目前頁/總頁數頁
```

## 9. 列印或輸出 PDF

設定完成後，點選右上角：

- `列印 / 另存 PDF`

即可：

- 直接列印
- 另存為 PDF

建議列印設定：

- 紙張：A4
- 方向：直式
- 縮放：預設或 100%

## 10. 本版重點

這一版為「自學筆記」第二版草稿，重點如下：

- 首頁固定標題為 `自學筆記`
- 頁面內標題預設為 `大南六甲`
- 支援動態欄數與列數
- 支援不同每頁格數
- 第一頁與後續頁有不同呈現規則
- 適合做自學、多格練習與列印筆記紙
