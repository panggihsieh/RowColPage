# 第四版操作手冊

本文件說明「第四版：克隆原始數學題目」的使用方式。

## 1. 開啟第四版

雲端網址：

- <https://panggihsieh.github.io/RowColPage/v4/>

若要在本機測試，可先於專案根目錄啟動靜態伺服器：

```powershell
python -m http.server 8000
```

再於瀏覽器開啟：

```text
http://localhost:8000/v4/
```

## 2. 第四版用途

第四版適合處理含有題號的數學圖片或 PDF，流程重點如下：

- 先上傳圖片或 PDF
- 根據題號規則自動辨識每一題起點
- 依題目順序裁出題圖
- 抽題後直接從第一題開始逐題確認
- 逐題確認裁切結果，必要時手動微調裁切框
- 確認裁切後，再執行 AI 辨識題目文字
- 若文字有誤，可手動修改後儲存
- 每儲存一題後自動跳到下一題
- 全部題目確認完成後，再開啟新的勾選頁
- 將勾選到的題目依序帶入左側題目區（題目文字 + 題圖）
- 右側保留計算程序區供列印後書寫

## 3. 版面設定區

可調整欄位如下：

- 標題
- 姓名
- 日期
- 起始編號
- 頁數
- 列數
- 每格輔助線
- 家長簽名

補充：

- 第四版版型固定為每題一列
- 左側為題目區
- 右側為計算程序區

## 4. 來源檔案抽題設定

第四版的核心設定在「來源檔案抽題設定」區。

### 來源檔案

上傳一份或多份來源檔案，可為：

- PDF
- 圖片

前提是檔案內要看得到題號。

測試檔案可直接使用專案內的 `v4/example.pdf`。

### 標題規則

每行輸入一條規則，可用：

- RegExp
- 關鍵字

例如：

- `^[（(]\s*[）)]\s*\d+\s*[.．、]`
- `^\d+\s*[.．、]`
- `^第\s*\d+\s*題`
- `^[（(]\d+[）)]`

系統會把符合規則的文字行視為每題起點。第四版的勾選流程會以這些題號作為選題依據。

### 範圍設定

- 左界 (%)：題目區塊左邊界
- 右界 (%)：題目區塊右邊界
- 標題上方留白：標題上方要多保留多少空間
- 頁尾保留：每頁底部要保留多少空白，不納入最後一題
- 下一題前間隔：在下一題標題前先留出一些空間，避免裁到下一題

## 5. 抽題流程

建議操作順序如下：

1. 上傳 PDF
2. 設定標題規則
3. 視 PDF 版面調整左右界與留白數值
4. 點選 `分析來源並顯示縮圖`
5. 逐題點開縮圖，確認裁切是否正確
6. 若裁切有誤，在放大視窗拖拉裁切框手動修正
7. 點選 `確認裁切並 AI 辨識`
8. 檢查辨識出的題目文字，必要時手動修改
9. 點選 `確認文字並儲存此題`
10. 系統會自動開啟下一題，重複同樣流程
11. 全部題目都儲存完成後，按 `全部確認完成，前往下一頁`
12. 在新的勾選頁勾選要保留的題目
13. 按下 `套用勾選結果`
14. 回到主頁確認左側題目區是否已依序填入

若題數超過目前頁數，系統會自動補足頁面。

補充：

- 每題都會先停留在主頁縮圖清單，尚未完成確認前不能前往勾選頁
- 放大視窗中可拖曳裁切框本體移動範圍，也可拖拉四角手動調整
- AI 辨識後的題目文字可直接修改，儲存後才算完成該題確認
- 勾選頁中每題都會同時顯示原始頁面縮圖與已確認的題圖

## 6. 抽題後的微調

即使已經完成自動抽題，仍可逐題手動調整：

- `上傳題圖`
- `更換題圖`
- `貼上題圖`
- `清除`

因此第四版可以先用來源檔案自動分題，再透過勾選頁挑題，最後只對少數題目做人工修正。

## 7. 列印與 PDF

設定完成後，點選：

- `列印 / 另存 PDF`

列印時會自動：

- 使用 A4 直式
- 保留左側題圖
- 保留右側計算程序區
- 隱藏上傳控制列
- 隱藏題號

## 8. 使用建議

若要提高抽題準確度，建議：

- PDF 中每題標題格式一致
- 題目盡量依固定版面排列
- 標題與題目內容之間不要混入太多裝飾元素
- 若題目裁太多，先縮小左右界或頁尾保留
- 若題目裁不完整，增加標題上方留白或下一題前間隔

## 9. 第四版完成重點

第四版目前完成的核心功能如下：

- 上傳含有題號的數學圖片或 PDF
- 依標題規則辨識題目起點
- 依空間範圍自動裁出題目區塊
- 支援逐題手動調整裁切框
- 支援 AI 辨識題目文字並手動修正
- 透過新的勾選頁挑選題目
- 將勾選到的題目依序填入左側題目區（文字 + 題圖）
- 保留手動逐題修正能力
- 列印成可書寫的原題數學複習單
