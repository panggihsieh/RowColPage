# 第三版操作手冊

本文件說明「第三版：幾何練習簿」目前完成版的使用方式。

## 1. 開啟第三版

雲端網址：

- <https://panggihsieh.github.io/RowColPage/v3/>

若要在本機測試，建議在 `v3` 目錄啟動靜態伺服器：

```powershell
cd v3
python -m http.server 8002
```

再於瀏覽器開啟：

```text
http://localhost:8002
```

## 2. 第三版版型說明

第三版採用固定的幾何練習版型：

- A4 直式
- 每題一列
- 左側為題目區
- 右側為計算程序區

每一列都對應一題，方便列印後直接在右側書寫解題過程。

## 3. 上方設定區

可調整欄位如下：

- 標題
- 姓名
- 日期
- 起始編號
- 頁數
- 列數
- 每格輔助線
- 家長簽名

補充：

- 第三版不提供欄數切換
- 每格輔助線目前支援：橫線、點、無
- 預設輔助線為：無

## 4. 題圖帶入方式

每一題左側題目區都可獨立放入圖片，支援以下方式：

### 方式一：上傳題圖

1. 點選 `上傳題圖`
2. 選擇幾何題目截圖
3. 圖片會自動顯示於左側題目區

### 方式二：更換題圖

若該題已有圖片：

1. 點選 `更換題圖`
2. 選擇新圖片
3. 新圖片會直接覆蓋原有圖片

### 方式三：Ctrl+V 貼上截圖

1. 先在其他軟體中複製截圖
2. 回到第三版頁面
3. 點選要貼上的目標格
4. 按 `Ctrl+V`

若剪貼簿中有圖片，系統會直接貼入該格。

### 方式四：貼上題圖按鈕

1. 先複製截圖
2. 點選該題的 `貼上題圖`
3. 系統會直接從剪貼簿讀取圖片

若瀏覽器未允許讀取剪貼簿，可改用 `Ctrl+V` 或一般上傳方式。

## 5. 題圖顯示規則

帶入圖片後：

- 題圖會自動縮放到左側題目區內
- 不會超出題目框
- 可重複更換圖片
- 可使用 `清除` 移除該題圖片

若該格是目前貼上目標，題目區外框會顯示提示效果。

## 6. 計算程序區

每題右側固定為計算程序區，主要用途如下：

- 列印後手寫計算
- 保留直式書寫空間
- 搭配輔助線進行解題

## 7. 列印與 PDF

設定完成後，點選：

- `列印 / 另存 PDF`

列印時會自動套用：

- A4 直式
- 題圖保留
- 計算程序區保留
- 上傳按鈕不顯示
- 題號不顯示

## 8. 建議使用方式

建議題圖內容：

- 幾何題目截圖
- 線段圖
- 角度圖
- 面積題
- 周長題

建議圖片整理方式：

- 盡量裁成只保留題目區域
- 避免過多空白邊
- 解析度不要太低

## 9. 第三版完成重點

第三版目前完成的核心功能如下：

- 逐題配置幾何題目圖片
- 左題目、右計算程序的固定版型
- 單題獨立上傳、更換、清除
- 剪貼簿貼圖流程
- 列印輸出為可書寫練習單

目前可作為正式使用的第三版版本。
