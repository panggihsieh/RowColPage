# 第三版操作手冊

本文件說明「第三版：幾何練習簿」的使用方式。

## 1. 開啟第三版

建議在 `v3` 目錄啟動本地伺服器：

```powershell
cd v3
python -m http.server 8002
```

再於瀏覽器輸入：

```text
http://localhost:8002
```

## 2. 第三版版型說明

第三版目前採用固定版型：

- A4 直式
- 固定兩欄版型
- 每一列一題
- 左邊為題目區
- 右邊為計算程序區

每一題的排列方式為：

- 左側：上傳題圖
- 右側：空白計算區

## 3. 上方設定區

可調整的欄位如下：

- 標題
- 姓名
- 日期
- 起始編號
- 頁數
- 列數
- 每格輔助線
- 家長簽名

補充：

- 欄數已固定，不提供切換
- 每格輔助線目前支援：橫線、點、無
- 預設為：無

## 4. 上傳題圖

每一列左側題目區左上角都有：

- `上傳題圖`

操作方式：

1. 點選 `上傳題圖`
2. 選擇一張幾何題目截圖
3. 圖片會自動顯示在左側題目區

## 5. 上傳後的顯示規則

上傳完成後：

- 題圖會自動縮放到左側題目框內
- 預覽畫面中，上傳按鈕會隱藏
- 題號在一般預覽畫面仍可看到

## 6. 更換題圖

若需要修改圖片：

1. 將滑鼠移到已上傳圖片的題目區
2. 顯示 `更換題圖`
3. 點選後重新上傳新圖片
4. 新圖片會直接覆蓋舊圖片

## 7. 計算程序區

每題右側固定為：

- 空白計算程序區

用途：

- 可列印後手寫計算
- 可搭配不同輔助線模式

## 8. 列印與 PDF

設定完成後，點選：

- `列印 / 另存 PDF`

列印時會自動套用：

- A4 直式
- 題圖保留
- 計算程序區保留
- 上傳按鈕不顯示
- 題號不顯示

## 9. 使用建議

建議題圖內容：

- 幾何題目截圖
- 線段圖
- 角度圖
- 面積題圖
- 周長題圖

建議圖片：

- 盡量裁成只保留題目區域
- 避免過多空白邊
- 解析度不要太低

## 10. 第三版重點

第三版目前的核心目的為：

- 逐題上傳幾何題目
- 左題目右計算程序
- 方便列印成可書寫的幾何練習單
