# 自學筆記

這是「自學筆記」第二版草稿的 Web App，使用 A4 直式版面，支援依欄數與列數動態產生不同格數的練習頁。

此版本已從第一版的固定八格簿，調整為可自訂每頁格數的「多格簿(電腦版)」草稿版，適合課堂練習、自主筆記與列印使用。

## 版本資訊

- 專案名稱：自學筆記
- 版本顯示：第二版草稿 / 多格簿(電腦版)
- 開發分支：`codex/v2-draft`
- 紙張格式：A4 直式
- 編號順序：左到右、上到下

## 第二版重點

- 每頁格數不再固定八格
- 可用「欄數 × 列數」動態決定每頁格數
- 第一頁顯示頁首資訊
- 第二頁之後只保留格子與頁碼
- 支援家長簽名顯示與隱藏
- 支援瀏覽器自動記住設定
- 支援一鍵重設預設值

## 目前固定規則

- 首頁固定主標題：`自學筆記`
- 頁面內建標題預設值：`大南六甲`
- 第一頁顯示：
  - 標題
  - 版型
  - 姓名
  - 日期
  - 家長簽名
- 第二頁之後：
  - 不顯示頁首資訊
  - 只顯示格子與頁碼
- 頁碼格式：`第目前頁/總頁數頁`
  - 例如：`第1/2頁`

## 可否在本地端執行

可以。

這個專案是純前端靜態網頁，沒有後端服務相依，因此可直接在本地端執行。

### 方式一：直接開啟檔案

直接用瀏覽器開啟 `index.html` 即可。

### 方式二：使用 localhost

在專案目錄執行：

```powershell
python -m http.server 8001
```

再用瀏覽器開啟：

```text
http://localhost:8001
```

如果要和第一版同時比較，建議第二版使用 `8001`。

## 目前功能

- 自訂頁面標題
- 設定班級
- 設定姓名
- 設定日期
- 設定起始編號
- 設定頁數

  - 橫線

## 介面說明

- 標題：設定第一頁標題與頁面內第一頁標題顯示
- 班級：目前保留在設定區，可作為後續擴充欄位
- 姓名：顯示在第一頁資訊列
- 日期：顯示在第一頁資訊列
- 起始編號：設定第一格的起始數字
- 頁數：設定產生幾頁
- 欄數：設定每頁有幾欄
- 列數：設定每頁有幾列
- 每格輔助線：切換格子中的輔助樣式
- 家長簽名：切換第一頁右上角簽名區顯示與隱藏
- 重設預設值：清除瀏覽器已記住的設定並回到預設狀態
- 列印 / 另存 PDF：輸出目前畫面

## 動態格數規則

第二版使用：

```text
每頁格數 = 欄數 × 列數
```

例如：

- `2 欄 × 4 列 = 8 格`
- `3 欄 × 5 列 = 15 格`
- `1 欄 × 2 列 = 2 格`

格子編號一律依照：

- 左到右
- 上到下

## 家長簽名區

若開啟「家長簽名」功能：

- 第一頁右上角會顯示兩列文字：
  - 家長
  - 簽名
- 右側顯示簽名框

若關閉：

- 第一頁簽名區隱藏

## 如何避免列印時多出一頁

這個專案先前曾發生列印時多出一頁的問題，因此目前版面採用以下原則：

- `.page` 固定 A4 高度
- `.grid` 使用剩餘空間分配
- 格子高度不寫死固定毫米
- 依欄列數動態套用 `grid-template-columns` 與 `grid-template-rows`

建議每次改版後都實際開啟列印預覽確認：

- 第一頁是否正常
- 第二頁之後是否只剩格子與頁碼
- 不同欄列數是否仍維持在單張 A4 內
