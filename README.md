# 八格簿

這是「八格簿」第一版的 Web App，適合用於國小數學或書寫練習。

目前版本提供 A4 直式排版、每頁 8 格、4 列 2 欄的版面配置，格子編號依照左到右、上到下自動排序，並可直接列印或另存 PDF。

## 版本資訊

- 版本名稱：八格簿
- 版本定位：第一版
- 版面格式：A4 直式
- 格數配置：每頁 8 格
- 排列方式：4 列 2 欄
- 編號順序：左到右、上到下

## 目前功能

- 自訂頁面標題
- 設定起始編號
- 設定頁數
- 切換每格輔助線
- 輔助線模式支援：
  - 橫線
  - 十字線
  - 點
  - 無
- 顯示或隱藏家長簽名區
- 頁尾中央顯示頁碼
- 支援列印與另存 PDF

## 介面說明

- 標題：可修改每一頁上方顯示的名稱
- 起始編號：設定第一格的編號起點
- 頁數：設定要產生幾頁
- 每格輔助線：切換格子中的練習輔助樣式
- 家長簽名：切換右上角家長簽名框的顯示與隱藏
- 列印 / 另存 PDF：輸出目前設定內容

## 檔案結構

- `index.html`：頁面結構與控制元件
- `styles.css`：A4 排版與畫面樣式
- `app.js`：互動邏輯、頁面產生與編號規則
- `README.md`：專案說明文件
- `操作流程.md`：操作步驟文件

## 啟動方式

### 方式一：直接開啟

直接用瀏覽器開啟 `index.html` 即可使用。

### 方式二：使用 localhost

在專案目錄執行：

```powershell
python -m http.server 8000
```

再用瀏覽器開啟：

```text
http://localhost:8000
```
