# RowColPage

RowColPage 是一組可列印的 A4 練習簿 Web App，目前包含三個版本：

- 第一版：八格簿
- 第二版：多格簿 / 自學筆記
- 第三版：幾何練習簿

整個專案採用純前端靜態網頁實作，可直接在瀏覽器開啟，也可透過本機伺服器或 GitHub Pages 使用，再列印或另存 PDF。

## 版本一覽

### 第一版：八格簿

特色：

- A4 直式
- 固定八格
- 編號由左到右、由上到下
- 支援多種輔助線
- 適合作為基礎數學作業簿

### 第二版：多格簿 / 自學筆記

特色：

- A4 直式
- 可自訂欄數與列數
- 每頁格數不再固定為八格
- 第一頁顯示頁首資訊，後續頁只保留格子與頁碼
- 支援家長簽名顯示切換

### 第三版：幾何練習簿

第三版已完成目前規劃的核心流程，主打「逐題整理幾何題目 + 右側保留手寫計算區」。

特色：

- A4 直式
- 單欄逐題排列，每頁依列數決定題數
- 每題左側為題目區，右側為計算程序區
- 支援標題、姓名、日期、起始編號、頁數、列數設定
- 支援家長簽名顯示切換
- 支援每題獨立上傳圖片
- 支援已上傳圖片的更換與清除
- 支援先複製截圖，再點選目標格後按 `Ctrl+V` 貼上
- 支援按下「貼上題圖」按鈕，直接從剪貼簿帶入圖片
- 題圖會自動縮放到題目區內
- 列印時自動隱藏題號與上傳控制列，只保留排版內容
- 每格輔助線支援：橫線、點、無
- 預設輔助線為：無

## 本地端執行

### 第一版

```powershell
python -m http.server 8000
```

開啟：

```text
http://localhost:8000
```

### 第二版

啟動根目錄靜態伺服器後，開啟：

```text
http://localhost:8000/v2/
```

### 第三版

建議在 `v3` 目錄單獨啟動：

```powershell
cd v3
python -m http.server 8002
```

開啟：

```text
http://localhost:8002
```

## 第三版使用重點

第三版目前建議流程如下：

1. 設定標題、姓名、日期、起始編號、頁數、列數。
2. 點選欲放入題圖的目標格。
3. 使用以下任一方式帶入圖片：
   - `上傳題圖`
   - `更換題圖`
   - `貼上題圖`
   - 直接按 `Ctrl+V`
4. 確認左側題目區圖片與右側計算程序區排版是否正確。
5. 若需要調整，重新更換圖片或清除該題。
6. 點選 `列印 / 另存 PDF` 輸出。

更完整的第三版說明，請參考：

- [第三版操作手冊](第三版操作手冊.md)

## 為什麼列印時可能多一頁

專案早期曾遇到「只加了一點頁首內容，就多出一頁」的情況。原因通常是：

- 頁面總高度仍以舊公式計算
- 但頁首或控制內容已增加
- 導致整體內容超過 A4 高度
- 瀏覽器列印時就會自動分到下一頁

目前的做法是：

- 讓 `.page` 固定為 A4 高度
- 讓 `.grid` 使用剩餘空間自動分配
- 避免每格高度用硬編碼公式計算

之後若再新增頁首欄位或題目區，仍要注意整頁總高度不能超出 A4。
