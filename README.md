# RowColPage

這個專案是一組可列印的 A4 練習簿 Web App，目前包含三個版本：

- 第一版：八格簿
- 第二版：多格簿 / 自學筆記
- 第三版：幾何練習簿

專案採用純前端靜態網頁實作，可直接在瀏覽器開啟，也可在 localhost 執行，再列印或另存 PDF。

## 版本一覽

### 第一版：八格簿

特色：

- A4 直式
- 固定八格
- 編號由左到右、由上到下
- 支援多種輔助線
- 適合作為基礎數學作業簿

網址：

- GitHub Pages：<https://panggihsieh.github.io/RowColPage/>

### 第二版：多格簿 / 自學筆記

特色：

- A4 直式
- 可自訂欄數與列數
- 每頁格數不再固定為八格
- 第一頁顯示頁首資訊，後續頁只保留格子與頁碼
- 支援家長簽名顯示切換

網址：

- GitHub Pages：<https://panggihsieh.github.io/RowColPage/v2/>

### 第三版：幾何練習簿

第三版以「逐題上傳幾何題目截圖」為主，版型重點如下：

- A4 直式
- 固定兩欄版型
- 設定區只保留列數，不提供欄數切換
- 每一列只對應一題
- 每題左邊是題目上傳區
- 每題右邊是空白計算程序區
- 題目截圖上傳後會自動縮放到左側題目框內
- 預覽時可滑鼠移入題目區再更換題圖
- 列印預覽時不顯示上傳按鈕與題號
- 每格輔助線目前支援：橫線、點、無
- 預設輔助線為：無

第三版目前建議先在本地端測試。

## 本地端執行

### 第一版

```powershell
python -m http.server 8000
```

開啟：

```text
http://localhost:8000
```

### 第二版

啟動根目錄靜態伺服器後，開啟：

```text
http://localhost:8000/v2/
```

### 第三版

第三版目前在 `v3` 目錄中，建議單獨啟動：

```powershell
cd v3
python -m http.server 8002
```

開啟：

```text
http://localhost:8002
```

## 第三版使用方式

第三版目前的實際流程如下：

1. 設定標題、姓名、日期、頁數、列數。
2. 每題左上角都有上傳題圖按鈕。
3. 逐題上傳幾何題目截圖。
4. 題圖會自動縮放並顯示在左側題目區。
5. 右側固定保留空白的計算程序區。
6. 若要改圖，可在預覽畫面滑鼠移到題目區後，使用「更換題圖」。
7. 列印時只保留排版內容，不顯示上傳控制與題號。

## 為什麼列印時可能多一頁

專案早期曾遇到「只加了一點頁首內容，就多出一頁」的情況。原因是：

- 頁面總高度仍以舊公式計算
- 但頁首高度已增加
- 導致整體內容超過 A4 高度
- 瀏覽器列印時就會自動分到下一頁

目前的做法是：

- 讓 `.page` 固定為 A4 高度
- 讓 `.grid` 使用剩餘空間自動分配
- 避免每格高度用硬編碼公式計算

之後若再新增頁首欄位或題目區，仍要注意整頁總高度不能超出 A4。
